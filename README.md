# LA Hotel Website

## Project Overview

The **LA Hotel Website** is a modern and responsive hotel website designed to showcase the services, facilities, and overall experience offered by LA Hotel. The website provides visitors with essential information about the hotel, including its location, amenities, and brand values, while maintaining a clean and user-friendly design.

This project was built to demonstrate front-end web development skills using standard web technologies and can be extended into a full hotel booking platform in the future.

##  Features

* Clean and modern user interface
* Responsive design for desktop and mobile devices
* Home page with hotel highlights and features
* About page describing LA Hotel’s brand and values
* Testimonials section from guests
* Well-structured and easy-to-maintain code

## Technologies Used

* **HTML5** – Page structure and content
* **CSS3** – Styling and layout
* **Font Awesome** – Icons for visual enhancement

## Getting Started

To view the website locally:

1. Download or clone the project files.
2. Open the `index.html` file in any modern web browser.
3. Ensure you have an internet connection to load external libraries (Font Awesome CDN).
